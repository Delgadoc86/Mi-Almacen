# Términos y Condiciones — Descarga e instalación de Mi Almacén

**Última actualización:** 06/07/2026

Este documento está pensado para publicarse en la página del sitio web
donde se ofrece la descarga de la aplicación "Mi Almacén"
(delgadodev.com.ar/proyectos/mi-almacen), junto al enlace de descarga. Es
**complementario** a los otros dos documentos ya redactados —
`TERMINOS_Y_CONDICIONES.md` (uso de la app una vez instalada) y
`POLITICA_DE_PRIVACIDAD.md` (datos que recolecta la app) — y cubre
específicamente lo que el sitio web y la descarga directa (fuera de Google
Play) requieren.

Al descargar y/o instalar la aplicación desde este sitio, el usuario acepta
estos términos.

- Esta versión de Mi Almacén **no está publicada en Google Play Store**.
  El archivo de instalación (.apk) se distribuye directamente desde este
  sitio web, alojado en Google Drive.
- **Google Drive es utilizado únicamente como servicio de alojamiento del
  archivo.** El Desarrollador no tiene ninguna afiliación con Google LLC ni
  con Google Play más allá de usar Firebase (backend) y Drive (hosting del
  instalador) como proveedores de infraestructura. Google no participa en

- El único canal oficial de descarga es este sitio web
  (**www.delgadodev.com.ar**). El Desarrollador no se responsabiliza por
  copias del instalador obtenidas de cualquier otro sitio, grupo de

## 3. Instalación desde una fuente desconocida (Android)

Al no distribuirse por Google Play, el sistema operativo Android va a
requerir que el usuario habilite manualmente la instalación de
aplicaciones de "fuentes desconocidas" (u "origen no confiable", según la
versión de Android) para poder instalar el archivo. Esta es una medida de
seguridad estándar del propio sistema operativo, no un indicio de que la
aplicación sea insegura.

- El usuario reconoce que habilita esa instalación **bajo su propio
  criterio y responsabilidad**, tras haber verificado que descarga el
  archivo desde el sitio oficial indicado en la Sección 2.
- El Desarrollador recomienda desactivar nuevamente esa opción del sistema
  luego de instalar la aplicación, como buena práctica general de
  seguridad del dispositivo (no específica de Mi Almacén).
